Spidy Math Battle

Interactive Math Learning Game with AI Integration

Fun Project AI-Generated Educational

About This Project

Spidy Math Battle is an engaging educational game created with Google AI Studio. Players answer math questions to help Spidy defeat enemies in an interactive combat system.

How It Works

  • Answer Questions: Solve math problems correctly to gain an advantage
  • Attack Enemies: Correct answers trigger Spidy's attacks on enemies
  • Health System: Wrong answers cost you one health bar
  • Progressive Difficulty: Questions get harder as you advance
  • AI-Powered: Questions generated by Google AI Studio

Technologies Used

Google AI Studio JavaScript HTML5 CSS3
